1. Make sure you have the lastest GNU C++ compiler, G++ and is working, if you are on windows, there is an option to install G++ when installing Cygwin
2. Open up the terminal and cd to the folder with rps.cpp
3. type "g++ -std=c++11 -Wall rps.cpp -o rps" without the quotations to compile the C++ file
4. Type "./rps <k value>" to run the code, k indicating number of results you would like the computer to look back at, without the quotations or carrots e.g ./rps 4
5. Type in 0, 1, or 2 as rock paper scissors
